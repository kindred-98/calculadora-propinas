# Asistencia de IA – Calculadora de Propinas

## 1. Uso de Inteligencia Artificial

Durante el desarrollo del proyecto se utilizó asistencia de IA para:

- Mejorar la estructura modular del proyecto.
- Implementar arquitectura híbrida (CLI + GUI).
- Generar ejecutable mediante PyInstaller.
- Mejorar validaciones y manejo de errores.
- Añadir documentación profesional con docstrings.

---

## 2. Preguntas realizadas a la IA

- ¿Cómo estructurar un proyecto Python modular?
- ¿Cómo convertir una aplicación Python en ejecutable?
- ¿Cómo escribir docstrings correctamente en Python?
- ¿Qué información debe incluir un buen docstring?
- ¿Cómo detectar errores en PyInstaller?
- ¿Cómo mejorar la arquitectura del proyecto?

---

## 3. Decisiones técnicas tomadas

- Separación en módulos (src/).
- Separación de responsabilidades (services, validators, gui).
- Implementación de arquitectura híbrida.
- Manejo profesional de excepciones.
- Documentación siguiendo estándar tipo Google.

---

## 4. Reflexión

El uso de IA permitió:
- Acelerar resolución de errores.
- Comprender problemas de empaquetado.
- Mejorar calidad del código.
- Aplicar buenas prácticas profesionales.

La lógica, implementación y pruebas fueron realizadas y comprendidas por el desarrollador.
