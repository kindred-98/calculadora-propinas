# Calculadora de Propinas

Proyecto educativo para desarrollar una calculadora de propinas interactiva en Python,
siguiendo buenas prácticas de programación, control de versiones con Git y asistencia de IA.

## Objetivos
- Calcular propinas con porcentaje fijo o personalizado.
- Dividir la cuenta entre varias personas.
- Validar entradas del usuario.
- Crear una interfaz interactiva en terminal.
- Documentar el proceso de desarrollo y asistencia de IA.

## Estructura del proyecto
- src/: Código fuente.
- docs/: Documentación adicional.
- requirements.txt: Dependencias del proyecto.
